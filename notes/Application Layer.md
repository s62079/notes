## Protocols

- domain: DNS
- host: DHCP
- email: SMTP, POP, IMAP
- file: FTP, TFTP
- web: HTTPS

### Email

- STMP: send mail to server
- POP: get mail from server, __then deletes it from server__
- IMAP: get mail from server

### File

- SMB: file sharing protocol for Microsoft networking, long term connection
